# Stock-Predictor
TISB Hackathon '24 Project

# Usage
Train a model:
```sh
$ python3 trainer.py
```
It will save a model in the current directory (`model.keras`)

Test/Run the model:
```sh
$ python3 tester.py
```
It will load the model from the current directory
